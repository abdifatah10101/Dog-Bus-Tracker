##  Features
-  **Seat-based system** using nested dictionaries  
-  Stores each dog's **name, breed, pickup & dropoff times**  
-  **Adds a new dog** if the bus has space  
-  **Removes a dog early** using `.pop()`  
-  **Prints starting & final rosters**  
-  Helps practice **Python data structures**
